# Click Fling

Roblox-TS source for the click fling script.

## Files

- `src/clickfling.client.ts` - source
- `include/` - roblox-ts runtime output used by Rojo
- `default.project.json` - Rojo project

## Build

```sh
npm install
npm run build
```

Release the compiled Luau from `out/clickfling.client.luau`.
